getVal(obj, str)

Look up a key's value when given the path as a string mimicing dot-notation. Used recreate the user object with the correct keys and proper nesting before inserting the document This allows for customizing the user document structure as desired.

Parameters:
Name Type Description
obj Object

Object to query. Almost always the signup object
str String

String representation of path. E.g., 'some.nested.path' or 'top_level_path'
